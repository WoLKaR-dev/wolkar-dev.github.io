VARIABLES

Una variable es un nombre que contiene valores. Los valores de las variables pueden ser numeros, strings, booleans, data types, y demás. Puedes usar cualquier nombre, pero no debe ser uno que comienze por un número, o que directamente sea el nombre de una funcion como if.
Aquí tienes un ejemplo: 

        
LETRAS   -- valido 
a1        -- valido
var_name  -- valido
_test     -- valido
if     -- NO valido
25th  -- NO valido

Es importante destacar que los nombre son case-sensitive, por tanto, VARIABLE y Variable son variables y nombres diferentes.

DEFINIR Y NOMBRAR VARIABLES

Para definir las variables usamos una funcion o no dependiendo del contexto. Si queremos que la variable aplique para todo el código simplemente escribimos el nombre, pero si sólo queremos que aplique dentro de una funcion en específico usamos la palabra local.
Aunque de todas formas, una función local va a tener el mismo efecto general en el código si se sitúa fuera de una funcion.
Aquí tienes otro ejemplo:

        
--Variable definida a todo el código:


Nombre = 'Oscar'  -- Variable general llamada Nombre
local Nombre = 'Oscar' -- Variable local llamada Nombre

NOMBRES RESERVADOS

Los nombres reservados son palabras que no se pueden usar para nombrar una variable, pues estas palabras representan una funcion. La lista de palabras reservadas son:

  • and
  • for
  • or
  • break
  • function
  • repeat
  • do
  • if
  • return
  • else
  • in
  • then
  • elseif
  • local
  • true
  • end
  • nil
  • until
  • false
  • not
  • while
